Facebook OAuth Direct Test

This page tests Facebook OAuth directly to identify any configuration issues.

Step 1: Test OAuth Authorization

Test Facebook OAuth

Expected Results:

  • If Working: Facebook login dialog appears, after login redirects to your callback
  • If Broken: "This app isn't available" or "needs at least one supported permission" error

Troubleshooting:

If you still see the error after configuring Facebook Login:

  1. Wait 5-10 minutes for Facebook's systems to update
  2. Try the test link in an incognito/private browser window
  3. Clear Facebook cookies and try again
  4. Ensure app is in "Live" mode, not "Development" mode

Manual Verification:

Check these URLs in Facebook Developer Console:

  • Facebook Login Settings
  • Basic Settings
  • App Review
